# 文章署名添加

给文章末尾添加署名。

**参数**：`$ARGUMENTS`（文章路径）

---

## 工作流程

1. 读取文章
2. 根据文章的 `tags` 和 `category` frontmatter 生成 hashtags
3. 在文章末尾添加署名块
4. 保存

## 署名格式

```markdown
---

**Aaron**

*ConnectOnion 创始人*

[当天日期]，悉尼

ConnectOnion 是目前最简单的 AI Agent 框架，原生支持自然语言作为协议

#标签1 #标签2 #标签3
```

**规则**：
- 日期用当天实际日期，格式：`2026年4月6日`
- hashtags 从文章 frontmatter 的 `tags` 字段提取，转为 `#标签` 格式
- 如果文章已有署名块，替换而不是追加
- 不修改文章正文内容
- **署名放在正文结尾之后、参考资料/Sources 之前**。如果文章有 `## 参考资料`、`## Sources`、`## References` 等章节，署名块插在该章节之前，不是文件最末尾。署名属于正文的收尾，参考资料属于附录。
